# FabCon 2025 Day 1: Copilot and AI Now Available for All Paid SKUs

FabCon 2025, held in Las Vegas, announced a game-changing decision: Copilot and AI features are now available across ALL paid Fabric SKUs, starting at F2 capacity.

**Previously:** Copilot required F64+ capacity (enterprise deployments)
**Now:** Available at F2 and above (half capacity, starting at $360/month)

**Impact:**
- 300x more organizations can access AI features
- Democratizes advanced analytics
- No premium tier lock-in for AI capabilities

**New Features Announced:**
- Agentic AI for autonomous analytics
- Azure AI Foundry integration
- Custom AI agents for business domains
- OneLake-wide unified security model
- Cross-tenant DevOps support

**OneLake Security Breakthrough:**
One of the most significant announcements was the new OneLake security model:
- Row-level security defined at OneLake level
- Automatically enforced in Spark, SQL, and Power BI
- No need to maintain separate security rules
- Massive simplification for governance

**Developer-Focused Announcements:**
- Fabric CLI now open source
- REST APIs expanded significantly
- Terraform support for IaC
- Improved CI/CD pipelines
