# Microsoft Fabric is Generally Available: Ignite 2023 Announcements

After months of public preview and refinement, Microsoft Fabric achieved General Availability status on November 15, 2023, at Microsoft Ignite. This marked a significant milestone: Fabric is now production-ready with full Microsoft support and SLAs.

**What GA Means:**
- SLA guarantees: 99.9% availability committed
- Production support: Full Microsoft support desk access
- Pricing available: Organizations can now purchase licenses
- No breaking changes: Microsoft commits to backward compatibility

**Fabric GA Architecture Components:**

**1. Data Engineering (Lakehouse)**
- Apache Spark for distributed processing
- Delta Lake table format for structured data
- Support for Python, PySpark, and Scala notebooks
- Automatic compute scaling based on workload

**2. Data Integration (Data Factory)**
- 600+ connectors for data sources
- Copy Activity for batch data movement
- Dataflows Gen2 for visual ETL/ELT
- Pipeline orchestration with 40+ activity types

**3. Data Warehousing**
- SQL Analytics Endpoint on Lakehouses
- Dedicated Synapse Data Warehouse
- Support for standard T-SQL operations
- Cross-warehouse querying

**4. Real-Time Analytics**
- Eventstreams for ingesting real-time data
- KQL (Kusto Query Language) databases
- Real-time dashboards and visualizations
- Event-driven automation with Activator

**5. Data Science & ML**
- Fabric Data Science workload
- MLflow integration for experiment tracking
- AutoML capabilities
- Model deployment and management

**6. Power BI Integration**
- Direct Lake connectivity to Lakehouses
- Semantic models in Fabric workspaces
- Real-time dashboards on streaming data
- Copilot for natural language analytics

**Governance & Security (GA Features):**
- Microsoft Purview integration
- Sensitivity labels and DLP
- Audit logging and compliance
- Workspace-level access control
- Row-level security (RLS)

**Capacity & Pricing:**
- Capacity units (CUs) determine compute performance
- Pay only for what you use (no per-user licensing)
- Starts at F2 (half capacity) for 360 CUs/month
- Scales to F2048 for enterprise deployments

**Known Limitations at GA:**
- Cross-workspace querying limited
- Some AI features in preview
- Admin REST APIs not yet complete
- Mirroring limited to SQL Server (other sources coming)

**Impact for Organizations:**
Fabric's GA status enables:
- Consolidation of multiple expensive tools
- Simplified licensing and cost management
- Unified governance and security
- Faster time-to-insight with integrated tools
